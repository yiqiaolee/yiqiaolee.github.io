Using Method on Yaml
2017/11/15 08:06:55 249

使用Yaml时，如果想要换行，在同一个： 下想写多行字符，比如，想要达到的效果是：
Tel： 1234
E—mail： xxx

那yml文件里的格式是这样的：(也就是用 | 保留换行符）
str： |
Tel: 1234
E-mail: xxx

这里要注意Tel： 跟E-mial 前面的空格是一样多的，都是比str那行多两个空格。
这个小小的符号，我居然花了1个多小时才测试好的…但是最后发现一个问题，为什么两行的时候可以，但是三行的时候第三行跟第二行就又到同一行了？？
