Projects 2017-11-16

Personalized Recommender Systems Research(PRSR) Guiyang,China
Sept.2015–present

  • Brief Description: PRSR leverages Game Theory as well as the “User-Pursue-Novel-Degree (UPND)” and “Music-Popularity (MP)” metrics to improve the performance of recommendation systems, especially to improve the personalized, real-time recommendation ability and mitigate the “Interest Drift” issue as well. This project regards the relationship between customers and system as a game relationship. This project focus on both off-line recommendation and on-line recommendation.
  • Core work:
    (1) Discussed with other workmates and defined two new novel metrics;
    (2)Explored the Game theory, Agent method, and Automatic Simulation Framework;
    (3) Proposed a novel music recommendation model based on personalized measurement and game theory.

Collaborative Filtering Recommender Systems Optimization (CFRSO) Guiyang,China
Dec.2015–present

  • Brief Description: CFRSO is a project aiming at optimizing collaborative filtering recommendation ability and improve its performance. This project focus on methods to mitigate the limitations and issues in collaborative filtering recommendation, such as low recommendation accuracy, time-consuming and low efficiency, especially facing with the trends of complex and high-dimensional big data.
  • Core work:
    (1) Investigated the current related approaches in terms of collaborative filtering optimization;
    (2) Matered the knowledge of NMF and E 2 LSH;
    (3) Proposed a new collaborative filtering recommender system model by leveraging the NMF and E 2 LSH.

Digital Intelligent Medical Assistant System (DIMS) Guiyang,China
Mar.2017–Sept.2017

  • Brief Description: DIMS integrates the computer and database management technology with the medical real
    requirements to develop an efficient, stable and easy-use assistant system which provides high quality of service
    to both doctors and patients.
  • Core work:
    (1) Discussed and analyzed the system requirements with workmates:
    (2) Designed parts of system front-end;
    (3)Implemented the back-end functions of system with ASP.NET.

Domestic Service Management System (DSMS) Guiyang,China
Sept.2016–Jan.2017

  • Brief Description: DSMS utilizes Netbeans as the developmental tool, MySQL as the back-end database system, and Java as the developmental language. It provides customers with services such as searching and making appointments. It also provides administrator with services such as employee management and customer
    management.
  • Core work:
    (1) Analyzed and documented the system requirements;
    (2) Implemented the database management with SQL server.

SunnyChaser Team Management System (SCTMS) Guiyang,China
Oct.2015–Jan.2016

  • Brief Description: SCTMS has three components, including Service Tasks Aggregation (STA), Transaction Tasks Aggregation(TSA), and Projects Management(PM). STA will allocate API tasks to project members located at the back-end. TSA will emit transaction and modification messages; PM will presents the relationship between each project and each project member.
  • Core work:
    (1) Designed the back-end database with MySQL;
    (2) Implemented and extended the database operations with Sqlalchemy.

Alumni System of Northwest Minzu University (ASNMU) Lanzhou,China
May,2013–May,2015

  • Brief Description: ASNMU is a project under the 2013 National University Student Innovation and Entrepreneurship Training Project. ASNMU develops a system to provide services for alumni of Northwest Minzu University, such as communication, connection, and donations.
  • Core work:
    (1) Wrote project proposal;
